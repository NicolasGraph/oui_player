h2. Bandcamp player config

h3. Recognised values for the @play@ attribute

* album=[*id*]
* track=[*id*]
* album=[*id*]/track=[*id*] 
* http(s)://bandcamp.com/(EmbeddedPlayer/)(album=[*id*]/)track=[*id*]

h3. Player size

- @width@ :=
The width of the player.
*Default:* related plugin pref
*Initial:* @350px@
*valid:* @<length>@ =:
- @height@ :=
The height of the player.
*Default:* related plugin pref
*Initial:* @588px@
*valid:* @<length>@ =:
- @ratio@ :=
The ratio used to calculate the missing value for the width or the height.
*Default:* related plugin pref
*Initial:* none
*valid:* integer[@:@]integer =:
