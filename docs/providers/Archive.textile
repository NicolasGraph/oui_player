h4. Archive additional attributes

Recognised values for the @play@ attribute:

* *id*
* http(s):(www.)//archive.org/details/[*id*]
* http(s):(www.)//archive.org/embed/[*id*]

Most of the following attributes can be set via the _Preferences_ tab; +always prefer plugin "global" prefs to tag related attributes!+
More informations on "this Archive webpage":https://archive.org/help/video.php.
